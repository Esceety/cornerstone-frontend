# Service Page Image Requirements

## Image Location
All service/program hero images should be placed in:
```
public/images/services/
```

## Image Specifications

### Youth Athletics Page
- **Filename**: `youth-athletics-enrichment-intervention.jpg`
- **Dimensions**: 1920px × 1200px (16:10 aspect ratio)
- **Format**: JPG or WebP
- **File Size**: Keep under 500KB (optimize for web)
- **Content**: Youth engaged in athletics, performing arts, or educational activities
- **SEO Alt Text**: Already configured in code

## Optimization Guidelines
- **Resolution**: 72 DPI (web standard)
- **Color Space**: RGB (not CMYK)
- **Compression**: Use tools like TinyJPG or ImageOptim
- **Responsive**: Images automatically scale on mobile devices
- **Loading**: First image uses priority loading for performance

## SEO Best Practices
- Filenames use descriptive, hyphenated keywords
- Alt text is comprehensive and descriptive
- Images are optimized for fast loading
- Modern formats (WebP) supported with JPG fallback

## Quick Action
1. Get/create your image (1920×1200px)
2. Optimize it for web (under 500KB)
3. Rename to: `youth-athletics-enrichment-intervention.jpg`
4. Drop it in: `cornerstone-frontend/public/images/services/`
5. Done! The page will automatically display it.
